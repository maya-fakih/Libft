# Libft
milestone 1 of 42 Beirut core

This project serves to help students understand the logic and algorithms behind standard functions everyone uses wether in c languages or in higher based ones like python...
This project is a good start if you want to understand the foundations of coding which can be a great starting point for nontecnical people wanting to learn about coding and development, as well as university students or junior devs who want to know more about the how does my code actually work the way it does. 
Follow along this journey to rediscover the tech world together :)
